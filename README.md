# s3-img-uploader

> Simple cli module for easily upload images to S3


## Install

```
$ npm install --save s3-img-uploader
```


## Usage

```sh
cat foo.html | s3-img-uploader --root-dir ./assets --s3-bucket some-bucket --s3-prefix some/prefix
```
